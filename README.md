# CIS 613 - Software Testing

This repository contains course contents, and assignments for **CIS 613: Software Testing**.  


## Course Overview
Discussion of the major techniques of software testing: software technical reviews, software testing techniques, proofs of correctness, and simulation/prototyping. Concludes with guidelines on organizational implications of software verification and validation activities.
